# 🌲 CatBoost "Под капотом": Реализация математики градиентного бустинга с нуля

Этот репозиторий содержит пошаговый разбор и ручную реализацию алгоритма градиентного бустинга на Python. Цель проекта — заглянуть внутрь библиотеки CatBoost и показать, как именно алгоритм строит деревья, считает сплиты и обновляет вероятности.

В файле [`PocketCatboostStudy.ipynb`](PocketCatboostStudy.ipynb) представлена полная реализация логики обучения, предсказания которой совпадают с оригинальным `CatBoostClassifier`.

## 🛠 Ключевые особенности (Features)

*   **Кастомный движок бустинга:** Написаны функции для прохождения по деревьям (`get_leaves_masks`), поиска лучших разбиений (`find_best_split`) и обновления предсказаний (`update_predictions`).
*   **Точная математика CatBoost:** Воспроизведена оригинальная формула метрики `L2` для поиска сплитов с учетом регуляризации (`l2_leaf_reg`).
*   **Метод Ньютона (Newton step):** Реализован расчет значений в листьях с использованием градиентов и гессианов.
*   **Динамическая обрезка (Pruning):** Алгоритм останавливает построение ветки, если новый сплит не улучшает базовый L2-score.
*   **100% совпадение:** Финальная валидация сравнивает результаты предсказаний с результатами CatboostClassifier.

## 📊 Структура экспериментов

Проверка математики проводится на двух наборах данных:

1.  **Игрушечный датасет (Кредитный скоринг):** 
    Набор данных из 15 наблюдений для предсказания дефолта (`loan_status`) на основе дохода (`person_income`) и долговой нагрузки (`loan_percent_income`)[cite: 1]. Позволяет визуально отследить работу каждого правила разбиения (например, `loan_percent_income <= 0.255` & `person_income <= 105000.0`)[cite: 1].
2.  **Классический датасет Titanic:** 
    Проверка движка на реальных, более сложных данных с использованием признаков `pclass`, `age`, `sibsp`, `parch` и `fare` для предсказания выживаемости (`survived`)[cite: 1].
